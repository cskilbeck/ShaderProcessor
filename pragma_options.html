Pragma Options

Pragmas must start on the 1st column of a line and cannot contain line breaks.

They're specified like this:

#pragma name(options)

See below for the list of pragma names.

The blend pragma must specify one or more optional indices (all channels specified will be assigned).

Options are a series of name = value settings.The default value for enum types is emphasized.

bool options can be specified as 'enabled' or 'disabled' or 'enabled = [true | false]' or 'disabled = [true | false].'

Anti-aliased line drawing enable/disable

#pragma anti_aliased_line(options)

Option Input
enabled bool (default = false)

Back Stencil operations and comparison function

#pragma back_stencil(options)

Option Input
fail_op decr decr_sat incr incr_sat invert keep replace zero
depth_fail_op decr decr_sat incr incr_sat invert keep replace zero
pass_op decr decr_sat incr incr_sat invert keep replace zero
comparison_func always equal greater greater_equal less less_equal not_equal

Culling mode & winding order (specifies the order of front-facing triangles)

#pragma culling(options)

Option Input
mode back front none
order ccw cw

Enable/Disable depth buffer reading & writing

#pragma depth(options)

Option Input
enabled bool (default = true)
write disabled enabled
comparison_func always equal greater greater_equal less less_equal not_equal

Depth Bias values

#pragma depth_bias(options)

Option Input
bias int (default = 0)
clamp float (default = 0.000000)
slope_scaled_bias float (default = 0.000000)

Depth Clipping enable/disable

#pragma depth_clip(options)

Option Input
enabled bool (default = true)
